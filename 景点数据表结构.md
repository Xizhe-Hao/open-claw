## 🏛️ 景点数据表结构 (Attractions Table Schema)

### 📊 数据库表设计

```sql
-- 景点主表 (attractions)
CREATE TABLE attractions (
    id VARCHAR(36) PRIMARY KEY DEFAULT (UUID()),
    name VARCHAR(200) NOT NULL,
    chinese_name VARCHAR(200),
    description TEXT,
    category ENUM('landmark', 'museum', 'park', 'entertainment', 'shopping', 'cultural', 'nature') NOT NULL,
    
    -- 位置信息
    address VARCHAR(500),
    latitude DECIMAL(10, 8),
    longitude DECIMAL(11, 8),
    district VARCHAR(100),
    city VARCHAR(100) DEFAULT 'Chicago',
    state VARCHAR(50) DEFAULT 'Illinois',
    country VARCHAR(50) DEFAULT 'USA',
    
    -- 营业信息
    opening_hours JSON, -- 存储每天的开放时间
    best_time_to_visit ENUM('morning', 'afternoon', 'evening', 'anytime'),
    estimated_duration_minutes INT,
    is_free BOOLEAN DEFAULT FALSE,
    
    -- 门票价格
    adult_price DECIMAL(10, 2),
    child_price DECIMAL(10, 2),
    senior_price DECIMAL(10, 2),
    student_price DECIMAL(10, 2),
    currency VARCHAR(3) DEFAULT 'USD',
    
    -- 评分与评价
    rating DECIMAL(3, 2) CHECK (rating >= 0 AND rating <= 5),
    review_count INT DEFAULT 0,
    popularity_score INT DEFAULT 0,
    
    -- 实用信息
    website_url VARCHAR(500),
    phone_number VARCHAR(50),
    email VARCHAR(200),
    
    -- 特色标签
    tags JSON, -- 存储标签数组如 ["family-friendly", "photography", "historic"]
    must_see BOOLEAN DEFAULT FALSE,
    seasonal_attraction BOOLEAN DEFAULT FALSE,
    
    -- 多媒体
    main_image_url VARCHAR(500),
    gallery_urls JSON, -- 图片URL数组
    video_url VARCHAR(500),
    
    -- 元数据
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    is_active BOOLEAN DEFAULT TRUE,
    version INT DEFAULT 1
);

-- 创建索引
CREATE INDEX idx_attractions_category ON attractions(category);
CREATE INDEX idx_attractions_district ON attractions(district);
CREATE INDEX idx_attractions_rating ON attractions(rating DESC);
CREATE INDEX idx_attractions_must_see ON attractions(must_see);
CREATE SPATIAL INDEX idx_attractions_location ON attractions(latitude, longitude);
```

### 🗂️ 相关数据表

#### 1. 景点特色表 (attraction_features)
```sql
CREATE TABLE attraction_features (
    id VARCHAR(36) PRIMARY KEY DEFAULT (UUID()),
    attraction_id VARCHAR(36) NOT NULL,
    feature_type ENUM('activity', 'facility', 'service', 'highlight'),
    name VARCHAR(200) NOT NULL,
    description TEXT,
    icon_url VARCHAR(500),
    is_available BOOLEAN DEFAULT TRUE,
    additional_cost DECIMAL(10, 2) DEFAULT 0.00,
    FOREIGN KEY (attraction_id) REFERENCES attractions(id) ON DELETE CASCADE
);
```

#### 2. 景点评论表 (attraction_reviews)
```sql
CREATE TABLE attraction_reviews (
    id VARCHAR(36) PRIMARY KEY DEFAULT (UUID()),
    attraction_id VARCHAR(36) NOT NULL,
    user_id VARCHAR(36) NOT NULL,
    rating INT CHECK (rating >= 1 AND rating <= 5),
    title VARCHAR(200),
    content TEXT,
    visit_date DATE,
    helpful_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    is_verified BOOLEAN DEFAULT FALSE,
    FOREIGN KEY (attraction_id) REFERENCES attractions(id) ON DELETE CASCADE
);
```

#### 3. 景点开放时间详情表 (opening_hours_detail)
```sql
CREATE TABLE opening_hours_detail (
    id VARCHAR(36) PRIMARY KEY DEFAULT (UUID()),
    attraction_id VARCHAR(36) NOT NULL,
    day_of_week INT CHECK (day_of_week >= 1 AND day_of_week <= 7), -- 1=Monday, 7=Sunday
    open_time TIME,
    close_time TIME,
    is_closed BOOLEAN DEFAULT FALSE,
    special_note VARCHAR(500),
    season_start DATE,
    season_end DATE,
    FOREIGN KEY (attraction_id) REFERENCES attractions(id) ON DELETE CASCADE
);
```

### 📈 数据统计表

#### 景点访问统计 (attraction_visits)
```sql
CREATE TABLE attraction_visits (
    id VARCHAR(36) PRIMARY KEY DEFAULT (UUID()),
    attraction_id VARCHAR(36) NOT NULL,
    date DATE NOT NULL,
    visitor_count INT DEFAULT 0,
    peak_hour TIME,
    average_duration_minutes INT,
    weather_condition VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY unique_attraction_date (attraction_id, date),
    FOREIGN KEY (attraction_id) REFERENCES attractions(id) ON DELETE CASCADE
);
```
