## 🚨 故障排除

### 常见问题

**1. 网站无法访问**
```bash
# 检查服务是否运行
ps aux | grep http.server

# 重启服务
pkill -f "http.server"
cd /home/ubuntu/clawd/chicago_travel_website
python3 -m http.server 8080 &
```

**2. 样式不加载**
- 检查CSS文件路径
- 清除浏览器缓存
- 检查控制台错误

**3. JavaScript不工作**
- 检查控制台错误
- 确保JavaScript已启用
- 检查文件加载顺序

**4. 图片不显示**
- 检查网络连接
- 检查图片URL
- 使用备用图片
